Variables

Todo lenguaje de programación tiene algo que se llama "variables". Las variables nos permiten almacenar tipos de datos que pueden ser usados luego e incluso modificados por nuestros algoritmos. El lenguaje C es un lenguaje de "tipos", eso quiere decir que se debe declarar que tipo de variable se va a utilizar antes de declararla.

Algunos tipos de variable en C son los siguientes:

int [Variable de tipo entero (Puede ir de los negativos al positivo)]

Por ejemplo int a = 5; o int b;. El primer ejemplo seria una variable inicializada y el segundo una variable no inicializada.

char [Caracteres imprimibles, estos deben ir delimitados por un vector]

Por ejemplo char c[5] = {'1', 'a', '2', 'b', '3'}; o char d[100];. El primer ejemplo seria una variable inicializada con un máximo de 5 elementos y el segundo una variable no inicializada con un máximo de 100 elementos.

float [Números flotantes o aquellos que poseen decimales]

Por ejemplo float e = 2.5 o float f;. De igual forma, una variable inicializada y otra no inicializada, en este caso incluye un decimal.

Imprimir variables por consola o pantalla

int Se imprime con %d
char Se imprime con %s o %c
float Se imprime con %f

Ejemplo:

#include <stdio.h>

int main(){
    int num1 = 5;
    char nombre[4] = {'J', 'O', 'S', 'E'};
    float num2 = 2.5;

    printf("El entero es %d \n", num1); //Se muestra con %d
    printf("El caracter es %s \n", nombre); //Se muestra el caracter
    printf("El flotante es %f", num2); // Se muestra el flotante
    return 0;
}

Introducir datos por consola o pantalla

Ahora sabiendo esto, tenemos un gran poder para mostrar resultados por pantalla, pero que tal si deseamos un método de introducción por el usuario. La libreria de input output nos permite introducir variables y funciona de forma parecida a la muestra de los datos utilizando scanf.

#include <stdio.h>

int main(){
    int a; //No requerimos inicializar una variable ya que el usuario la va a sobreescribir

    printf("Introduce un valor: ");
    scanf("%d", &a); //Sintaxis para leer enteros. Intenta hacerlo para flotantes

    printf("\n El valor que colocaste es: %d", a);
}

Para los caracteres seria..

#include <stdio.h>

int main(){
    char nombre[20]; //No requerimos inicializar una variable ya que el usuario la va a sobreescribir

    printf("Coloque su nombre: ");
    scanf("%s", nombre); //Sintaxis para leer caracteres

    printf("\n Un gusto en conocerte: %s", nombre);
}
