Switch-Case y Funciones

#Switch Case Las funciones Switch y Case nos permitiran realizar un estudio de casos que permitira ejecutar un bloque de codigos. Su funcion es parecida a un condicional ya que realiza una verificacion para cuya expresion sea cierta para poder ejecutar el bloque de codigo.

#Funciones en C Las funciones permiten al programador mantener un orden en el codigo y poder identificar facilmente que hace cada bloque de codigo. En C, cada funcion debe ser declarada con un tipo, bien sea int, char o float como hemos visto anteriormente o un tipo void que no espera retornar ningun resultado. Un ejemplo de una funcion es int main(){} que es la funcion principal, es de tipo entero y por ello espera retornar un valor; es por eso que al final colocamos un return 0.

Las funciones tienen otras propiedades que son los parametros o argumentos que es lo que colocamos dentro de los (). Los parametros o argumentos permiten pasar variables a la funcion y los parametros pueden ser todo tipo de variables. Para inyectar variables a una funcion se debe respetar el orden de los argumentos.

#include <stdio.h>
#include <stdlib.h>
//MENU CON FUNCIONES UTILIZANDO SWITCH CASE
void sumar(int numero1, int numero2){
    numero1 += numero2;
    printf("\n El resultado es :  %d", numero1);
}

void contar(int numero){
    for(int i = 0; i < numero; i++){
        printf("\n%d", i);
    }
}
int main(){
    int opcion = 0;
    int op;

    printf("\n1. Suma de dos numeros");
    printf("\n2. Contador recursivo");
    printf("\nColoque una opcion : ");
    scanf("%d", &opcion);

    switch(opcion){ //Abrimos el switch para hacer el estudio de casos
        case 1: int n1, n2; //Evaluamos para la opcion 1 de nuestro menu
                printf("\nColoque dos numeros : ");
                scanf("%d%", &n1);
                scanf("%d", &n2);
                sumar(n1, n2); //Aqui llamamos a la funcion sumar que esta declarada anteriormente. Se inyectan ambos valores y dicha funcion espera 2 parametros
                break; //Detenemos la ejecucion del bloque de codigo

        case 2: int n3; //Evaluamos para la opcion 2 de nuestro menu
                printf("\n Coloque un numero : ");
                scanf("%d", &n3);
                contar(n3);
                break;
    }

    printf("\n Desea continuar? Si(1), No(2)");
    scanf("%d", &op);
    if(op == 1){
        system("cls") //Limpiamos pantalla
        main(); //Volvemos a llamar a la funcion main para iniciar nuevamente nuestro menu
    }
    
    return 0;
}
