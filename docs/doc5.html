Operadores Matemáticos

En C como otros lenguajes de programación, podemos utilizar operaciones matemáticas básicas para resolver cualquier tipo de problema. Es trabajo del ingeniero entender y en muchos casos desarrollar las fórmulas más eficientes de cálculos para resolver dichos problemas. No te asustes! Estos operadores no pasan de ser lo más básico a nivel de matemáticas. Ejemplo de ello son los siguientes operadores:

Algunos operadores matemáticos

En este curso estaremos trabajando con los operadores:

+ suma, - resta, * multiplicación, / división, % módulo o residuo, = igualación

Ejemplo de un algoritmo que calcula el area de un rectángulo:

#include <stdio.h>

int main(){
    int lado1, lado2, resultado;
    printf("Introduzca el lado 1 y el lado 2 : ");
    scanf("%d", &lado1);
    scanf("%d", &lado2);

    resultado = lado1*lado2; //Igualamos resultado a la multiplicacion de lado1 * lado2
    printf("El resultado es: %d , para el lado 1 que equivale a %d y el lado 2 que equivale a %d", resultado, lado1, lado2);
}
