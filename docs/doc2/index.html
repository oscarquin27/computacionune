Entorno de desarrollo

Para comenzar, deberás descargar el programa Dev C/C++ . Es un compilador que te permitirá correr todo código de lenguaje C o C++. En los cursos estaremos compilando solamente en C, esto quiere decir que todos los archivos tendrán una extensión .h . Por ejemplo programa.h. Deberás descargar el programa de la siguiente pagina dev C/C++ Puedes irte familiarizando con tu nuevo entorno de desarrollo ya que ahí es donde se realizarán todas las prácticas.

Las funciones principales con las que trabajaremos en este compilador es con Compilar y ejecutar, tan sencillo como eso, debido a que el lenguaje C no es compilado a tiempo real, sin embargo al día de hoy sigue siendo una herramienta muy poderosa. Este compilador muestra todos los errores que tengas en compilación que se explicarán en la clase, se puede modificar también el color de fondo a preferencia entre otras aplicaciones.
