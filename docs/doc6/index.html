Condicionales

En programación, una sentencia condicional es una instrucción o grupo de instrucciones que se pueden ejecutar o no en función del valor de una condición. Los tipos más conocidos de sentencias condicionales son el SI..ENTONCES (if..then), el SI..ENTONCES..SI NO (if..then..else) y el SEGÚN (case o switch), aunque también podríamos mencionar al manejo de excepciones como una alternativa más moderna para evitar el "anidamiento" de sentencias condicionales.

En este curso estaremos trabajando con los operadores lógicos siguientes:

&& Compuerta AND, || Compuerta OR, ! Compuerta NOT, != NOT EQUAL

Ejemplo de un algoritmo que verifica si las edades de dos personas son mayores a 18 años:

#include <stdio.h>

int main(){
    int edad1, edad2;
    printf("Introduzca la edad 1 y 2: ");
    scanf("%d", &edad1);
    scanf("%d", &edad2);

    if(edad1 > 18 && edad2 > 18){
        printf("Ambas edades son mayores a 18 años");
    }

    return 0;
}

En este ejemplo solo verificamos si ambas edades son mayores a 18 años, pero si quisieramos saber cual es la otra posibilidad, en este caso, cuando ambas o al menos una de ellas no sea mayor de 18 podriamos hacer lo siguiente:

#include <stdio.h>

int main(){
    int edad1, edad2;
    printf("Introduzca la edad 1 y 2: ");
    scanf("%d", &edad1);
    scanf("%d", &edad2);

    if(edad1 > 18 && edad2 > 18){
        printf("Ambas edades son mayores a 18 años");
    }
    else{
        printf("Al menos una persona es menor a 18 años");
    }

    return 0;
}

Otra variante al mismo algoritmo podria ser el siguiente:

#include <stdio.h>

int main(){
    int edad1, edad2;
    printf("Introduzca la edad 1 y 2: ");
    scanf("%d", &edad1);
    scanf("%d", &edad2);

    if(edad1 > 18 && edad2 > 18){
        printf("Ambas edades son mayores a 18 años");
    }
    else if(edad1 < 18 || edad2 < 18){
        printf("Al menos una persona es menor a 18 años");
    }

    return 0;
}

Este es un resultado valido mas poco eficiente ya que no tiene mucho sentido realizar este tipo de validacion pues se sabe que la opcion que faltaba es que al menos una persona fuera menor de edad. Ahora, si quisieramos saber quien es la persona cuya edad es menor a 18 entonces seria:

#include <stdio.h>

int main(){
    int edad1, edad2;
    printf("Introduzca la edad 1 y 2: ");
    scanf("%d", &edad1);
    scanf("%d", &edad2);

    if(edad1 > 18 && edad2 > 18){
        printf("Ambas edades son mayores a 18 años");
    }
    else if(edad1 < 18){ //Verificamos si la Persona 1 es menor de edad
        printf("La persona 1 es menor a 18 años");  
    }
    else {  // En caso contrario que ninguna se aplique, la Persona 2 es menor de edad
        printf("La persona 2 es menor a 18 años"); 
    }

    return 0;
}

Los condicionales pueden ayudar a resolver problemas muy complejos en programación. Evidentemente ya que trabajamos con memoria, el conjunto de operacioens que podemos realizar por sentencia es limitado, pero rara vez se llega a ese límite. Aunque de ser asi se puede recurrir a métodos de simplificación mediante la lógica de computación, matemáticas discretas u otros mecanismos.

Vamos a un ejercicio más complejo en el cual podamos utilizar todas las compuertas:


/*Calcular si el area de un triangulo da como resultado un numero par, de no ser asi debe ser convertido
a un numero par si y solo si es mayor o igual a 10 o mayor a 1000, 
luego debe ser sumado a otro numero proporcionado por el usuario. */
#include <stdio.h>

int main(){
    int base, altura, otro, resultado;

    printf("\n Dame la altura y base de un triangulo: ");
    scanf("%d%d", &altura, &base);

    resultado = (base*altura)/2;

    if((resultado%2) != 0 && (resultado >= 10 || resultado > 1000)){
        resultado *= 2; //Convertido a numero par
        printf("\nEl par es: %d ", resultado); //Mostramos el par
        printf("\nDiga otro numero: ");
        scanf("%d", &otro);
        resultado += otro;
    }

    printf("El resultado es : %d", resultado);

    return 0;
}
