Ciclos FOR y WHILE

Los ciclos for y while permiten hacer iteraciones dentro de nuestro programa, es decir, hacer un cierto numero de repeticiones para facilitar un trabajo.

Por ejemplo : Si quisieramos imprimir por pantalla un numero 10 veces, es mucho mas sencillo hacerlo con una iteracion que de forma manual. Esto le ahorra tiempo al programador y muchas lineas de codigo ya que si alguna funcion tiene que ejecutarse miles de veces o esta ejecucion es variable, las iteraciones serian la unica solucion.

#include <stdio.h>
//Utilizando el for para contar del 0 - N 
int main(){
    int numero;
    printf("Introduzca un numero: ");
    scanf("%d", &numero);
    
    for(int i = 0; i <= numero; i++){
        printf("%d", i); //Imprimimos i que es la variable que aumenta
    }
    

    return 0;
}

#include <stdio.h>
//Utilizando el while para contar del 0 - N 
int main(){
    int numero;
    printf("Introduzca un numero: ");
    scanf("%d", &numero);
    
    while(i <= numero){
        printf("%d", i);
        i++;
    }

    return 0;
}
