Arrays o Vectores

Los Arrays son un tipo de dato que nos permiten almacenar un tipo de variable. En el caso de otros lenguajes de programacion mas permisivos como Javascript, un array puede contener varios tipos de datos.

Un array tiene varias propiedades y lo denotamos con el siguiente simbolo []. Las propiedades de un array son el valor,la posicion y el tamano. Las posiciones se cuentan en bits desde el 0 hasta el tamano - 1.

Ejemplo de un array : [1,2,3,4] Segun sus propiedades lo podemos desglosar de la siguiente forma: Tamano del array : 4 Posicion 0 - Valor - 1 Posicion 1 - Valor - 2
Posicion 2 - Valor - 3 Posicion 3 - Valor - 4

Los arrays son iterables, quiere decir entonces que podemos realizar funciones mas complejas gracias a este tipo de variables. Es sumamente importante aprender el uso de los vectores ya que es una herramienta poderosa para el desarrollador, en ambitos como la inteligencia artificial, machine learning, etc, los vectores y matrices son ampliamente utilizados.

#include <stdio.h>
//Declaracion de un vector
int main(){
    int v[5] = {1, 2 , 3 , 4 , 5} //Denotamos dentro del corchete el tamano del vector y luego dentro de {} los valores que deseamos
                                  //A este vector lo denotamos como un vector lleno

    printf("\nPosicion 0 , valor %d", v[0]); //Mostramos que valor se encuentra en la posicion 0
    printf("\nPosicion 0 , valor %d", v[1]); //Mostramos que valor se encuentra en la posicion 1
    printf("\nPosicion 0 , valor %d", v[2]); //Mostramos que valor se encuentra en la posicion 2
    printf("\nPosicion 0 , valor %d", v[3]); //Mostramos que valor se encuentra en la posicion 3
    printf("\nPosicion 0 , valor %d", v[4]); //Mostramos que valor se encuentra en la posicion 4

    return 0;
}

#Llenar y Mostrar un vector

Muchas veces el programador podria desconocer que valores se encuentran dentro de un vector, o podria decidir declarar un vector vacio para llenarlo despues. Esto permite dinamismo en el desarrollo y aumentar los niveles de complejidad.

#include <stdio.h>
#include <stdlib.h>
#define MAX 100  //Declaramos una constante que servira para indicar el tamano maximo que puede tener nuestro vector

int main(){
    int v[MAX]; //Colocamos dentro el tamano maximo que puede tener el array y al no ser inicializado lo declaramos como un vector vacio
    int n; 

    printf("\nIndique tamano del vector : ");
    scanf("%d", &n);

    for(int i = 0; i < n; i++){ 
        v[i] = rand(); //Esta funcion permite asignarle un valor pseudo aleatorio en la posicion de donde se encuentre el vector.
                       //Si se fijan la variable i arranca en 0 e ira aumentando hasta el tamano -1 del vector y habra llenado dicho vector. 
    }

    for(int i = 0; i < n; i++){
        if(i == (n -1)){          //Por cuestiones de estetica podemos decirle que si llego al final del vector no vamos a imprimir una flecha al final 
            printf("[%d]", v[i]);
        }
        else{
            printf("[%d] -> ", v[i]); //En este caso mostraremos el vector con esta iteracion
        }
    }
}
