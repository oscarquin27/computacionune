Introducción a lenguaje C

Esta documentación te proporcionará toda la información acerca de lenguaje C que se estará viendo a lo largo de todo el trimestre en las materias de Computación IV, V y VI.

Computación IV

Este curso abarcará todos los principios básicos de este lenguaje de programación, enfocándose a los condicionales para resolver todo tipo de problemas, así como también en los ciclos o iteraciones para generar programas con interacciones automáticas.

Computación V

El curso de computación V se centrará en las iteraciones, vectores y matrices con diferentes algoritmos para resolver problemas más complejos que se presenten en el mundo real. Se busca con este curso que el estudiante desarrolle toda la capacidad para resolver cualquier tipo de problema de programación en cualquier lenguaje.

Computación VI

Este curso se enfoca más a algoritmos complejos, aprovechando las estructuras de datos para tener soluciones más elegantes a distintos tipos de problemas. Este curso busca expandir los conocimientos del estudiante que le puedan brindar más oportunidades en el campo laboral.
