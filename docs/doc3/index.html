Hola Mundo

El primer programa de todo desarrollador es el famoso "Hola Mundo", en C es muy peculiar la forma de llegar hasta allí ya que de primeras se tiene que importar una librería con la que estaremos trabajando a lo largo del curso, que se llama stdio.h que es sencillamente una herramienta que nos permitir introducir y mostrar salidas de texto en consola.

La sintaxis para dicho programa sería el siguiente:

C
#include <stdio.h>

int main(){
    printf("Hello World");
    return 0;
}

Cómo entiende C el compilador este código?

Es muy sencillo:

    #include <stdio.h>

Se refiere a incluir la librería que se mencionó anteriormente

    int main(){}

Se refiere a la función principal. Todo programa en C debe tener una función principal

Ahora...

    printf("Hola Mundo");

Imprimirá en consola o pantalla Hola Mundo. Y finalmente...

    return 0;

Se encargará de terminar el programa. Después de un retun, ninguna sentencia después de él será ejecutada. De igual forma esto será entendido más adelante.

Imprimir más texto

Podemos incluir más texto en C simplemente llamando a más funciones printf y podemos utilizar \n para indicar un salto de línea. Por ejemplo:

#include<stdio.h>

int main(){
    printf("Hola Mundo \n");
    printf("Mi nombre es: ___ \n");
    printf("Hasta luego");
    return 0;
}

Este comando permitirá más adelante darle forma a diferentes programas que realicemos y darle un mejor aspecto visual que también nos ayudará a entender que es lo que estamos compilando.
